# 《两山20年》作品简介

## 作品概述

这是一部沉浸式数据新闻作品，以"绿水青山就是金山银山"理念提出20周年为背景，通过交互式网页叙事，展现中国生态文明建设的绿色变革历程。作品融合滚动驱动动画、数据可视化、3D交互等现代Web技术，将宏大主题转化为可感、可知、可互动的数字体验。

**核心命题：** 绿水青山如何成为金山银山？生态价值如何被看见、被计算、被实现？

---

## 章节结构

| 章节 | 页面 | 核心内容 |
|------|------|----------|
| 开篇·两山20年 | `首页/dist/index.html` | 余村前后对比，滚动切换"过去←→现在" |
| 壹·第一个坐标 | 同上#chapter-1 | 8张卡片完整讲述余村转型故事；附可交互收入变化图表 |
| 贰·青山为凭 | `三层时间变换/index.html#chapter-2` | 森林覆盖率四阶段演变（1990年16.74% → 2024年>25%） |
| 叁·碧空为证 | 同上#chapter-3 | PM2.5四阶段治理（2015年46μg/m³ → 2024年29.3μg/m³） |
| 肆·清流为鉴 | 同上#chapter-4 | 地表水水质四阶段改善（Ⅰ-Ⅲ类断面64.5% → 90.4%） |
| 伍·一棵树值多少钱？ | `首页/dist/wu.html` | 引入GEP概念，交互展示生态价值转化4条路径 |
| 陆·675+331，1006个绿色坐标 | `超卷轴/index.html` | 3D地图标注全国1006个生态文明建设示范区/两山基地 |
| 附·翻转立方体 | `翻转立方体/index.html` | 13个第一批两山基地故事，3D立方体交互展示 |
