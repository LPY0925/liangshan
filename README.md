# 两山20年 — 生态文化数字展馆

## 作品概述

沉浸式数据新闻作品，以「绿水青山就是金山银山」理念提出20周年为背景，通过交互式网页叙事，展现中国生态文明建设的绿色变革历程。

作品融合滚动驱动动画、数据可视化、3D交互等现代 Web 技术，将宏大主题转化为可感、可知、可互动的数字体验。

**核心命题：** 绿水青山如何成为金山银山？生态价值如何被看见、被计算、被实现？

## 章节结构

1. 首页 — 序章引入
2. 超级周 — 生态文明建设历程
3. 翻轉立方體 — 3D交互展示
4. 三层时间变换 — 历史对比
5. 日月轮转 — 生态循环
6. 次页打字机 — 叙事呈现

## 技术栈

- HTML / CSS / JavaScript
- 3D 交互效果
- 滚动驱动动画
- 响应式设计
